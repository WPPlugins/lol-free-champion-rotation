This plugin will add a widget that shows the current League of Legends free-to-play champions.

== Description ==

Every week there are 10 new League of Legends champions you can try for free. Our plugin lets you show their icons in any wordpress sidebar. We connect to the Riot Games API and we'll forward the data to all of you whenever we find changes.

The champion names and titles are now available in multiple languages:

* Czech (Czech Republic)	
* German (Germany)	
* Greek (Greece)	
* English (United States)	
* Spanish (Spain)
* French (France)
* Hungarian (Hungary)
* Indonesian (Indonesia)
* Italian (Italy)	
* Japanese (Japan)
* Korean (Korea)
* Malay (Malaysia)
* Polish (Poland)
* Portuguese (Brazil)
* Romanian (Romania)
* Russian (Russia)
* Thai (Thailand)
* Turkish (Turkey)
* Vietnamese (Viet Nam)
* Chinese (China)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to Appearance->Widgets and add LoL Free Champion Rotation to any sidebar
1. Enter a title and a width if you wish.
1. Hit save and you're done!
